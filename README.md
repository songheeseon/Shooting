# Shooting Game

# <H3> 이 게임은 유튜브 강의와 에셋스토어 에셋을 활용하여 만든 2D 종스크롤 슈팅게임입니다. </H3>
<h2> 1. 오브젝트 풀링 및 원근감 있는 무한 배경 구현, 보스 패턴 4가지 기능 구현 +@ 추가 패턴 혹은 랜덤으로 보스가 공격할 수 있도록 수정
<h2> 2. 사망 시, 펫 비행기도 비활성화 하는 기능 추가 구현 
<h2> 3. 레벨 디자인을 통해 스테이지 2 까지 구현 완료.
